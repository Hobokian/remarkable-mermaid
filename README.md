# remarkable-mermaid

A plugin that uses custom code fences in remarkable to create mermaid diagrams inside

## Installation

  npm install remarkable-mermaid --save

## Usage
### (Remove # from the front of code fences and remove all empty lines)

  #``` mermaid

  sequenceDiagram

  Alice->>Bob:Hello

  Bob->>Jim:Hi

  #```
