# remarkable-mermaid

A plugin that uses custom code fences in remarkable to create mermaid diagrams inside

## Installation

  npm install remarkable-mermaid --save

## Usage (Remove () from mermaid)

  var mermaidAPI = require('mermaidAPI');

  ``` # mermaid

  sequenceDiagram

  Alice->>Bob:Hello

  Bob->>Jim:Hi
  
  #```
