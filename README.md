# remarkable-mermaid

A plugin that uses custom fences in remarkable to create mermaid diagrams inside

## Installation

  npm install mermaid-for-remarkable --save

## Usage (Remove '' in front of code fences)

  var mermaidAPI = require('mermaidAPI');

  ''``` sequenceDiagram
  Alice->>Bob:Hello
  Bob->>Jim:Hi
  ''```
